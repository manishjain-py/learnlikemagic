EXPLANATION PRINCIPLES — follow all twelve:
1. Use the Simplest Words the Student Would Use — write as if explaining to a Grade {grade} student who struggles with this subject. Use words THEY would naturally say, not textbook words. Prefer "add" over "combine," "part" over "fraction" (on first use). If you must use a technical term, introduce it gently with a plain-language explanation right alongside it.
2. One Idea Per Card — each card teaches exactly one thing. If you need "and" to describe what a card covers, split it into two cards.
3. Start with What They Already Know — anchor every new idea to something the student already understands. Don't start with the definition — start with the experience.
4. Show, Don't Just Tell — use ASCII diagrams, number lines, tables, step-by-step worked examples, or formatted illustrations wherever a concept can be seen. Even simple visual representations help.
5. Examples Before Rules — present a concrete example first, then the general rule. Students remember examples; they forget rules stated in isolation.
6. Use Everyday Analogies — But Make Them Precise — good analogies make abstract concepts tangible. Every analogy must map correctly — if it breaks under scrutiny, don't use it.
7. Keep It Short — each card should be readable in 15-30 seconds. 12 short cards are better than 4 long ones. Brevity is clarity.
8. Build Progressively — No Leaps — each card should feel like a small, natural step from the previous one. If card 4 doesn't feel obvious after cards 1-3, there's a missing card.
9. Anticipate Misconceptions — address common wrong ways to think about the concept directly. Don't wait for confusion; prevent it.
10. End with Confidence, Not Complexity — the last card should make the student feel "I understand this." End with a clean summary, not an edge case.
11. Explain Like a Favourite Older Sibling, Not a Textbook — use a warm, casual, conversational tone, like a favourite older sibling who just "gets it" and wants to help. Address the student directly ("you", "let's", "notice how"). Be encouraging without being patronizing or over-the-top. No gamified hype ("Amazing! You're a SUPERSTAR!"). Use 0-1 emojis per card max. No ALL CAPS for emphasis. Keep it real — a calm, supportive voice that makes hard things feel doable.
12. Design Each Variant as a Fresh Perspective — this variant must feel like a different tutor, not a rephrased version of another approach.

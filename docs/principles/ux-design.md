# Principles: UX Design

Guiding principles for all user-facing interfaces. Primary audience = Grade 3-8 students on phones.

## 1. One Thing Per Screen

Each screen has one clear purpose. Big, clear buttons. No cognitive overload. If a screen does two things, split it.

## 2. Minimal Typing

Pickers and selectors over free-text input. Voice input available. Large tap targets. Kids shouldn't need to type paragraphs.

## 3. Warm Language

"What's your name?" not "Enter display name". "You're all set! Let's start learning" not "Account created successfully". Errors are kind, not scary.

## 4. Mobile-First

Designed for phone screens first. Large tap targets. Touch-friendly. Desktop is secondary.

## 5. Fast, Never Slow

Every interaction feels snappy. If something takes time, show progress. Never leave student staring at blank screen.

## 6. Skippable Optional Steps

Optional steps (enrichment profile, preferences) have clear "Skip for now". Students can start learning immediately. Personalization improves with data but doesn't block access.

## 7. Forgiving Input

Messy input handled gracefully. Don't reject, interpret. Don't punish, guide.

# Principles: Content Extraction from Books

How we extract teachable content from textbooks while respecting copyright.

## 1. Full Coverage — Every Concept Gets Taught

Extract every concept, skill, formula, and learning objective the chapter covers. A student using our platform should learn everything the textbook teaches — no gaps, no skipped sections. If the book covers it, we teach it.

## 2. Extract What to Teach, Not How the Book Teaches It

We extract the **subject matter** (topics, concepts, depth, sequencing) — never the book's **expression** (wording, explanations, examples, analogies, diagrams). The book tells us *what* a student needs to learn and *how deep*; our tutor creates its own way to teach it.

## 3. No Verbatim Content

Never copy sentences, worked examples, practice problems, illustrations, or any original expression from the book. Guidelines should describe the concept and target depth — not reproduce the author's text. The extracted content should be unrecognizable as derived from any specific book.

## 4. Depth Matches the Book's Intent

If the book spends 3 pages on long division with remainders and half a page on checking with multiplication, our topic should reflect that weighting. Extract the intended depth and emphasis — which concepts are core vs. supplementary — so the tutor allocates time proportionally.

## 5. Curriculum Alignment Over Book Loyalty

When a book's coverage is incomplete or idiosyncratic, align with the curriculum standard (board/grade). The book is our primary source for *what to teach*, but the curriculum is the authority on *what matters*.
